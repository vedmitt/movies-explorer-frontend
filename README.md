# movies-explorer-frontend

Фронтенд часть дипломного проекта по курсу "Веб-разработчик" от Яндекс практикума.

### Стек технологий:
- HTML
- CSS
- React
- MongoDB

## Чтобы запустить локально

1. установите зависимости:
`npm install` 

2. Затем запустите сервер:
`npm start`

3. Для сборки:
`npm build`
